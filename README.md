#  CareerVerse AI

CareerVerse AI is an interactive career simulation platform designed to help students explore and experience different professions before making career decisions.

Unlike traditional career guidance systems that simply recommend careers, CareerVerse immerses users in realistic workplace scenarios, decision-making challenges, skill requirements, and learning roadmaps. This enables students to understand the day-to-day responsibilities and realities of various professions through experiential learning.

## Features

* Interactive career simulations
* Product Manager career journey
* Software Engineer career journey
* Data Analyst exploration
* Business Analyst exploration
* Skill guidance and learning recommendations
* Career roadmaps for beginners
* AI-powered conversational interface
* Real-world workplace decision making

## Technologies Used

* Python
* Flask
* HTML5
* CSS3
* JavaScript
* Google Gemini API
* GitHub

## Objective

To help students make informed career decisions by allowing them to virtually experience professional roles before choosing a career path.

##  Run Locally

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
python app.py
```

Open:

```text
http://127.0.0.1:5000
```
